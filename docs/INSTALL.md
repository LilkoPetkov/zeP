## **docs/INSTALL.md**

## Installing zeP

### **Windows (PowerShell, as Administrator)**

```powershell
powershell -c "irm https://zep.run/install.ps1|iex"
```

### **Linux**

```bash
curl -fsSL https://zep.run/install.sh | bash
```

### **AUR**

```bash
yay -S zep-bin
```

### **macOS**

```bash
brew tap XerWoho/homebrew-zep
brew install zep
```
